# Santiago Gonzalez Portfolio

Professional portfolio built with React and prepared for GitHub Pages deployment.

## Highlights

- Modern single-page portfolio with a professional visual system
- Real profile data, direct contact links and downloadable CV
- Selected GitHub projects surfaced inside the landing page
- Responsive layout with working section anchors
- Updated public metadata for cleaner sharing and indexing

## Scripts

```bash
npm install
npm start
npm run build
```

## Deployment

The project is configured with:

- `homepage`: `https://sgonnzalezv3.github.io/Portafolio`
- static build artifact mirrored into `docs/`

If GitHub Pages is configured to serve from the `docs/` folder on `main`, pushing the updated repository is enough.
